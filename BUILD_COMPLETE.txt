================================================================================
SECURITY & LIMITATIONS
================================================================================

Limitations:
- Data stored locally (browser cache clears it)
- No real-time between browsers
- No backend validation (use backend for production)
- Passwords not hashed (use bcrypt for production)

This is suitable for:
✅ Learning and demos
✅ Prototyping
✅ Testing features
✅ Portfolio projects

For production, add:
- Real backend server
- Password hashing (bcrypt)
- HTTPS/SSL
- JWT tokens
- Rate limiting
- Input validation
